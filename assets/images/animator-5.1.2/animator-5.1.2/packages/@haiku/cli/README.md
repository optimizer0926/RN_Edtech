# @haiku/cli
  
> The Haiku CLI

## Installation

Via yarn:

```
$ yarn global add @haiku/cli
```

Via npm:

```
$ npm install -g @haiku/cli
```

## Usage

For a list of available commands, enter:

```
$ haiku --help
```
